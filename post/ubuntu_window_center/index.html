Ubuntu中让打开的程序窗口居中

2018-12-05 1 min read

每次在Ubuntu中打开新的窗口都会靠左上角显示，所以每次打开一个窗口都要多做一步操作，将窗口移到屏幕中间来，强迫症的我觉得太麻烦了，所以...

  1. 安装ccsm

    $ sudo apt-get install compizconfig-settings-manager

  2. 打开ccsm

    $ ccsm

  3. 选择**放置窗口，在安置模式处选择居中**
